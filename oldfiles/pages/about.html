Express Chess

Express Chess uses Vue for its front end design while implementing Node for a backend server. The server runs a Rest API using Express and Mongo for data storage.

All logic for valid movements are done in the back end, while the front end focuses on display and selection. It's still far from done, and some of the logic isn't flawless!

Check out the GitHub link at the bottom to see the code in the back end!
